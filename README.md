# The House Prototype

Standalone desktop prototype for the House.

## Run

```sh
npm install
npm run dev
```

The Vite UI is served at `http://127.0.0.1:5173`, and Electron opens it in a
standalone desktop window.

## Current Prototype

- Electron desktop shell.
- React/Vite/TypeScript UI.
- Two-floor blueprint with clickable rooms.
- Mock residents with animated state markers and local movement.
- Conversation and activity terminal tabs.
- Shared chat input that writes into the House conversation.
- Steward routing marker for user messages without spending a model call.
- One-on-one direct rooms from the terminal plus button.
- Top-level House and Population views.
- Top-level Steward diagnostics view for integrity, failure, consent, and source-state checks.
- Controlled room structure editor with file-backed room config writes.
- Dedicated Population UI with roles, activity, prompts, model selection, and permissions.
- Per-persona model selection metadata.
- Editable per-persona system prompts persisted in local runtime state.
- Direct rooms route through the selected persona model when `ANTHROPIC_API_KEY` is set.
- Local runtime clock with real-time, accelerated testing, and paused modes.
- Motion pause/resume, tick speed, and daily call budget controls.
- Activity feed generated by cheap local behavior rolls.
- Staggered persona decisions with downtime between actions.
- Movement intents with en-route state and arrival events.
- Room layout, furniture, items, and affordances in the room inspector.
- Room convergence trigger for social rooms.
- Active room conversation objects with participants and witnesses.
- Persona awareness settings and filtered direct-call context.
- Librarian JSONL memory v1 with file latest activity and recall query.
- Export/load buttons for file-backed persona, room, and runtime JSON snapshots.
- Backup creation/listing and selected-backup restore for `config` and `state`.
- File-backed House events, persona memories, relationship updates, and direct-room transcript snapshots.
- Source provenance diagnostics for memories and relationship updates.
- Placeholder Anthropic API boundary using `ANTHROPIC_API_KEY` when added later.

## Notes

The current runtime is local and cheap. Personas do not call a model yet, and
the House does not modify itself. Those systems are intentionally staged behind
the permission, backup, and stasis design docs.
